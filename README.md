# MyCode  
## 解脱之味不独饮,&emsp;开心之果不独证  
## 简介
`MyCode`是作者自己平时的学习积累，里面包含了各种基本算法，`LeetCode`刷题记录，`Java`基础知识点和各种笔记等.  
## 需要
* 基于 `JDK1.8`
* `Maven3.3.9`构建(`可选`). 
## 阅读    
读者可以在线观看也可以使用`git clone`拉取到本地阅读，本项目的所有程序都可以在`JDK1.8`，`Maven3.3.9`环境下编译通过，正常运行    
## 纠错
欢迎广大读者对于本项目不足之处批评指正，作者会在第一时间进行修正
