# MyCode  
## 解脱之味不独饮,开心之果不独证  
**MyCode是作者自己平时的学习积累，里面包含了各种基本算法，LeetCode刷题记录，Java基础知识点和各种笔记等.**  
**MyCode基于 JDK1.8，Maven3.3.9构建.**    
**读者可以在线观看也可以使用git clone 拉取到本地阅读，本项目的所有程序都可以在JDK1.8，Maven3.3.9环境下编译通过，正常运行**     
### 欢迎广大读者对于本项目不足之处批评指正，作者会在第一时间进行修正
